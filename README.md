# Password-Security-Sniffing-Session-Analysis

> ⚠️ This project was performed in a legal and authorized lab/testing environment only.

🔐 Task 3: Password Security, Sniffing & Session Analysis
📌 Overview

This project demonstrates network traffic analysis, password auditing, and session security concepts using a controlled lab environment. It includes packet capture using Wireshark, password cracking using John the Ripper/Hydra, and analysis of session hijacking risks.

🛠 Tools Used
1. Wireshark

Wireshark
Used for capturing and analyzing network traffic such as HTTP, DNS, TCP, and ARP packets.

2. John the Ripper

John the Ripper
Used for cracking password hashes using dictionary attacks.

3. Hydra

Hydra
Used for brute-force login attacks on services like FTP/HTTP (lab only).


🎯 Conclusion

This lab demonstrates how insecure network communication and weak authentication mechanisms can expose systems to serious security threats. Proper encryption, strong passwords, and secure session handling are essential for protection.
